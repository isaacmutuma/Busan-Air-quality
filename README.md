# Busan-Air-quality
NumPy project analysing simulated air quality sensor data across 8 sensors 
over 24 hours in Busan, South Korea.
< Still a work in progress >
## What it covers
- Data cleaning — error codes, NaN detection, outlier capping
- Statistical analysis — z-scores, sensor ranking, AQI scoring
- Distance matrix between sensors using broadcasting
- Linear regression to predict pollution trends per sensor
- Saving cleaned data to compressed and CSV formats

## Tools
Python · NumPy · Jupyter
